  2. list-style-type:none - 删除圆点。导航栏不需要列表项标记。

CSS 图像透明度

  1. 定义透明效果的 CSS3 属性是 opacity 
    img{opacity:0.4;filter:alpha(opacity=40); /* For IE8 and earlier */}
  2. IE9, Firefox, Chrome, Opera 和 Safari 使用属性 opacity 来设定透明度。opacity 属性能够设置的值从 0.0 到 1.0。值越小，越透明。
  3. IE8 以及更早的版本使用滤镜 filter:alpha(opacity=x)。x 能够取的值从 0 到 100。值越小，越透明。
  4. background-attachment:fixed 设置固定的背景图像
