CSS position 属性

  1. position:relative;left:-20px;表示相对于正常位置向左移动
  2. overflow 设置当元素的内容溢出时发生的事情overflow:auto;  overflow:hidden;  overflow:scroll;
  3. clip 属性剪裁绝对定位元素 
    img {position:absolute;clip:rect(0px 500px 200px 0px)}
  4. z-index 设置元素的堆叠顺序，z-index:-1位于当前元素的底部，z-index默认值为0，z-index：1在当前元素上部
  5. a:hover {background-color:#ff33ff}
  6. li {display:inline}将列表转为行内元素
  7. CSS属性选择器
    a[href] {color:red;}
  8. 与简单属性选择器类似，可以把多个属性-值选择器链接在一起来选择一个文档。请注意，这种格式要求必须与属性值完全匹配 
    a[href="http://www.w3school.com.cn/"][title="W3School"] {color: red;}
  9. 如果需要根据属性值中的词列表的某个词进行选择，则需要使用波浪号（~）。
    假设您想选择 class 属性中包含 important 的元素，可以用下面这个选择器做到这一点： 
    p[class~="important"] {color: red;}

下表是对这些选择器的简单总结：

类型 描述
[abc^="def"] 选择 abc 属性值以 "def" 开头的所有元素
[abc$="def"] 选择 abc 属性值以 "def" 结尾的所有元素
[abc*="def"] 选择 abc 属性值中包含子串 "def" 的所有元素

CSS 选择器参考手册

选择器 描述
[attribute] 用于选取带有指定属性的元素。
[attribute=value] 用于选取带有指定属性和值的元素。
[attribute~=value] 用于选取属性值中包含指定词汇的元素。
[attribute|=value] 用于选取带有以指定值开头的属性值的元素，该值必须是整个单词。
[attribute^=value] 匹配属性值以指定值开头的每个元素。
[attribute$=value] 匹配属性值以指定值结尾的每个元素。
[attribute*=value] 匹配属性值中包含指定值的每个元素。

后代选择器

  1. 有关后代选择器有一个易被忽视的方面，即两个元素之间的层次间隔可以是无限的。

Child seceltors

  1. 如果您不希望选择任意的后代元素，而是希望缩小范围，只选择某个元素的子元素，请使用子元素选择器（Child selector）。
  2. 子选择器使用了大于号（子结合符）。子结合符两边可以有空白符，这是可选的。因此，以下写法都没有问题： 
    h1 > strong
h1> strong
h1 >strong
h1>strong

相邻兄弟选择器（Adjacent sibling selector）

  1. 如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器
  2. 相邻兄弟选择器使用了加号（+），即相邻兄弟结合符（Adjacent sibling combinator）。
    3. h1 + p {margin-top:50px;}
  3. 用一个结合符只能选择两个相邻兄弟中的第二个元素

锚伪类

  1. a:link {color: #FF0000}		/* 未访问的链接 */
a:visited {color: #00FF00}	/* 已访问的链接 */
a:hover {color: #FF00FF}	/* 鼠标移动到链接上 */
a:active {color: #0000FF}	/* 选定的链接 */
  2. 在 CSS 定义中，a:hover 必须被置于 a:link 和 a:visited 之后，才是有效的。
  3. 在 CSS 定义中，a:active 必须被置于 a:hover 之后，才是有效的。
  4. 最常见的错误是认为 p:first-child 之类的选择器会选择 p 元素的第一个子元素.

伪元素

  1. "first-line" 伪元素用于向文本的首行设置特殊样式。
  2. first-line" 伪元素只能用于块级元素。
  3. :first-letter 伪元素元素用于向文本的首字母设置特殊样式
  4. ":before" 伪元素可以在元素的内容前面插入新内容。
  5. ":after" 伪元素可以在元素的内容之后插入新内容。
