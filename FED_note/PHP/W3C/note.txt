PHP语法
php脚本以<?php开头，以?>结束
php中，所有用户定义的函数、类和关键词都对大小写不敏感
不过所有变量都对大小写敏感
PHP变量
规则：
变量以$符号开头，其后是变量的名称
变量名称必须以字母或下划线开头
变量名称不能以数字开头
变量名称只能包含字母数字字符和下划线
变量名称对大小写敏感
变量会在首次为其赋值时被创建
变量作用域
函数之外声明的变量拥有 Global 作用域，只能在函数以外进行访问。
函数内部声明的变量拥有 LOCAL 作用域，只能在函数内部进行访问。
global关键词用于访问函数内的全局变量
static关键词能保留函数内某个局部变量

PHP echo print语句
echo能够输出一个以上字符串
print只能输出一个字符串，并始终返回1

PHP数据类型
字符串 整数 浮点数 逻辑 数组 对象 null
<?php
$x=5985;
var_dump($x);   //返回变量的数据类型和值
?>

PHP字符串函数
strlen()函数返回字符串的长度
strpos()函数用于检索字符串内指定的字符或文本
如果找到匹配则会返回首个匹配的字符位置，若未找到匹配则返回false
<?php
echo strpos("Hello world!","world");
?>

PHP常量
常量以字符或下划线开头（常量名称前面没有$符号）
常量贯穿整个脚本时自动全局的
如需设置常量可使用define（）函数
函数首个参数定义常量名称
第二个参数定义常量的值
第三个定义是否对大小写不敏感，默认false

PHP字符串运算符
.串接         $txt1 = "Hello" $txt2 = $txt1 . " world!" 
.=串接赋值    $txt1 = "Hello" $txt1 .= " world!" 
PHP数组运算符
+ 联合

PHP foreach循环
foreach循环只适用于数组，并用于遍历数组中的每个键值对

PHP创建用户自定义函数
函数名能够以字母或下划线开头
函数名对大小写不敏感
函数名 语义化 能够反映函数所执行的任务
PHP默认参数值 函数可添加默认参数

PHP数组
索引数组 带有数字索引的数组
关联数组 带有指定键的数组
$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");
foreach循环遍历关联数组
多维数组 包含一个或多个数组的数组

PHP数组排序
sort()以升序排列
rsort()降序排列
asort()根据值，以升序对关联数组排列
ksort()根据键，以升序对关联数组进行排序
arsort()根据值，以降序对关联数组排列
krsort()根据键，以降序对关联数组排列
count()函数用于获取返回数组的长度

PHP超全局
    $GLOBALS
这种全局变量用于在 PHP 脚本中的任意位置访问全局变量（从函数或方法中均可）。
    $_SERVER
这种超全局变量保存关于报头、路径和脚本位置的信息。
    $_REQUEST
用于收集 HTML 表单提交的数据。
    $_POST
广泛用于收集提交 method="post" 的 HTML 表单后的表单数据。$_POST 也常用于传递变量。
    $_GET
也可用于收集提交 HTML 表单 (method="get") 之后的表单数据。
    $_FILES
    $_ENV
    $_COOKIE
    $_SESSION

PHP表单处理
get方法从表单发送的信息对任何人都是可见的，信息数量也有限制
get可用于发送非敏感数据
post发送的信息是不可见的，且信息数量无限制

PHP表单验证
<form method="post" action="<?php echo htmlspecialchars($_SERVER["PHP_SELF"]);?>">
$_SERVER["PHP_SELF"]将表单数据发送到页面本身，而不是跳转到另一张页面，这样用户就能在表单页面获得错误提示信息
htmlspecialchars()函数把特殊字符转换为HTML实体，这样可防止攻击者通过在表单中注入HTML代码
